# Vehicle Loan Calculator 

Vehicle Loan Estimator is an Android application that helps users calculate vehicle loan payments easily. The app is designed with a simple and user-friendly interface to make loan estimation fast and accurate.

## Features

- Home navigation to enter:
  - Vehicle price
  - Down payment
  - Loan period
  - Interest rate
- Have calculation of:
  - Loan amount
  - Total interest
  - Total payment
  - Monthly payment
- About navigation page with:
  - Author information
  - GitHub profile link

## Built With

- Android Studio
- Java Programming Language
- XML for UI Design

## How to Run the Project

1. Clone this repository
2. Open it using Android Studio
3. Run the project on emulator or real device
